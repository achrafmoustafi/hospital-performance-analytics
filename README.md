# 📊 Project Overview

This project focuses on analyzing **hospital operational performance** using Business Intelligence tools.

The objective is to evaluate healthcare efficiency through key performance indicators such as:

- Average Length of Stay (LOS)
- Cost per Patient
- Patient Volume
- Department Performance

The project demonstrates how **data analytics and interactive dashboards** can support healthcare decision-making.
